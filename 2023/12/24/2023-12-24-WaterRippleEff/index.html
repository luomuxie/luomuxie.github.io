计算分辨率和UV

10. 是一个基础分辨率值。这个值是经验性的，用于确定波纹效果的基本大小和扩散程度。

exp2 函数计算2的幂。这里使用 -3.*iMouse.x/iResolution.x 作为指数。

  • iMouse.x 是当前鼠标的 x 坐标。
  • iResolution.x 是渲染窗口的宽度。
  • -3.*iMouse.x/iResolution.x 计算一个基于鼠标位置与屏幕宽度比例的值，然后乘以 -3。这个操作提供了一个用于调整波纹分辨率的指数。

exp2

整个表达式 exp2(-3.*iMouse.x/iResolution.x) 会根据鼠标的 x 位置动态调整波纹的密度。当鼠标向右移动时，波纹密度减小（因为指数变得更负），向左移动时增加。使用 exp2 可以产生更平滑或更强烈的调整效果， 它提供了一种非线性变化，这在视觉效果编程中很常见，用于创造更自然或更富有表现力的效果。

其中的常数10和-3都经验值

  • 10. 是一个基础分辨率乘数，这个值决定了波纹效果的基本大小和密度。调整这个值会影响波纹的整体尺度。
  • -3. 是另一个乘数，用于控制鼠标位置对分辨率的影响程度。更改这个值会改变鼠标移动对效果密度的影响强度。

计算UV及标准化坐标

fragCoord.xy / iResolution.y 将屏幕坐标标准化到 [0, 1] 范围内，这是一个常见的做法，用于将屏幕像素坐标转换为标准化的设备坐标（NDC）。

网络单元划分

floor 通对每个分量向下取整操作实现网格划分，使每个波纹都在自己的网格单元上做处理，每个网格单元都可以看作是波纹效果的一个独立起点或参考点。

理解这个过程：

  1. 坐标转换

    • uv 通常表示归一化的屏幕坐标，但在此之前它们已经乘以了一个分辨率参数，使得这些坐标可以超出 [0, 1] 的范围。

  2. 向下取整

    • 应用 floor 函数意味着对 uv 坐标的每个分量进行向下取整，这样每个坐标都被对齐到其所在网格单元的左下角。

网格单元划分的目的

  • 波纹效果的局部化：
    • 在后续的处理中，这样的网格划分使得波纹效果可以在每个单元内独立生成和演化。这样，即使是大范围的波纹效果，也能在每个网格单元内保持其独特性。
  • 性能优化：
    • 通过限制计算在离散的网格单元内进行，可以提高整体渲染效果的效率，特别是在处理大量数据时。
  • 视觉多样性：
    • 这种方法还允许在视觉上创造出更加丰富和多样化的效果，因为波纹在不同的网格单元中可以展现不同的阶段和特性。

结论

通过这种网格单元的划分，着色器能够以一种高效且有组织的方式处理复杂的视觉效果，如水面波纹。这不仅有助于性能优化，而且还为创造独特的视觉效果提供了基础

哈希函数分析

用于在着色器程序中生成伪随机数。这种哈希函数在计算机图形学中常用于生成看起来随机但实际上是确定的和可重复的值。这对于创建如噪声、纹理变化或者在您的例子中的波纹效果等视觉效果非常有用。

  • 输入：接收一个二维向量 vec2 p 作为输入。
  • 扩展和缩放：首先，p 被扩展成一个三维向量 p3（通过复制 p.x 到 p3 的 x 和 z 分量）。然后使用一个常量 HASHSCALE1 对 p3 进行缩放。这个缩放有助于“散布”输入值。
  • 分数部分：fract 函数用于获取缩放后值的小数部分，这是生成伪随机性的一个常见技术。
  • 点积运算：然后计算 p3 与其部分重排后的自身（p3.yzx）的点积，并加上常量（19.19），以进一步增强随机性。
  • 最终哈希值：通过对 p3 的前两个分量求和，然后再次取分数部分，生成最终的哈希值。这个值是一个介于0和1之间的浮点数。
为什么要用点积：

高哈希函数的随机性和均匀性

  1. 组合分量：点积将输入向量的各个分量结合起来，这有助于确保输入向量的所有部分都对最终结果有贡献，从而提高随机性。
  2. 生成单一值：点积计算结果是一个单一的标量值，这在哈希函数中非常有用，因为它可以用来进一步生成更复杂的、看似随机的模式。

时间因子和距离计算

动态计算波纹的扩散。它为每个像素点相对于波纹源的位置和波纹的生命周期创建了一个动态映射，从而产生了波纹随时间在水面上扩散的效果。

  • t 是基于时间 (iTime) 和哈希值计算的一个分数值，用于控制波纹随时间的变化。
  • v 是从当前波纹源到正在处理的像素点的向量。
  • d 表示从波纹源到当前像素点的距离，减去一个基于时间的变量，这决定了波纹扩散的速度和范围。

float(MAX_RADIUS) + 1. 是波纹的最大扩散半径，加1是为了确保波纹能够完全扩散到这个半径范围内。length(v) - (float(MAX_RADIUS) + 1.)*t 的结果是波纹扩散的动态半径。当 t 从0增加到1时，波纹从波纹源开始扩散，直到达到最大半径。

求波纹强度

通过两个接近点的波纹强度来近似求解波纹强度随空间的变化率（梯度），这是一种在实时图形渲染中常用的简化方法。通过这种方式，它有效地模拟了波纹随着时间和空间的动态变化，创建了逼真的水波纹效果。这种近似方法既高效又足够精确，适用于计算机图形学中的实时渲染需求。

  • p1 和 p2 计算了在 d1 和 d2 这两个距离处的波纹强度。
  • sin(31.*d1) 和 sin(31.*d2) 通过正弦函数创建周期性的波纹模式。
  • smoothstep 函数用于平滑波纹的边缘，使波纹逐渐消失，而不是突然中断。
为什么使用两个 p 值

  • 在物理和图形学中，了解一个量如何随空间变化是非常重要的。在这种情况下，我们想知道波纹强度如何随距离变化。

  • 计算梯度（即函数的导数）提供了这种变化率的信息。

  • 在连续数学中，导数是通过极限定义的，但在计算机图形学中，我们通常使用差分来近似导数。

  • 通过计算 p 在两个非常接近的点（d1 和 d2）的值，然后计算这两个值的差异，可以得到波纹强度关于距离的近似变化率。

  • 梯度近似：((p2 - p1) / (2. * h)) 这部分计算的是波纹强度的变化率，即梯度的近似值。

效果和用途
  • 这种方法允许着色器准确地模拟波纹随距离的变化情况，这对于创建逼真的波纹效果非常重要。
  • 梯度信息可以用来模拟光线如何与波纹相互作用，例如，通过影响波纹的亮度或颜色来增强视觉效果。
  • 在实时渲染中，这种数值梯度近似方法既高效又实用，因为它避免了复杂的数学计算。

总之，使用两个 p 值来计算波纹强度的近似梯度是一个在图形编程中常用的技巧，它能够有效地模拟波纹随空间变化的特性，从而为创建逼真的动态水波纹效果提供了必要的数学基础

累积波纹影响

  • 这里将计算出的波纹强度梯度累加到 circles 变量中。
  • 使用 normalize(v) 保证方向的一致性，((p2 - p1) / (2. * h)) 计算强度的梯度。
  • (1. - t) * (1. - t) 用于根据时间减少波纹的影响，使波纹随时间消退。单个 (1. - t) 提供了一个线性的衰减效果，即波纹强度随时间线性减小。然而，线性衰减可能不足以模拟水波纹随时间的自然衰减方式。通过使用 (1. - t) * (1. - t)，实际上是将衰减效果的曲线变得更加陡峭。这意味着波纹强度在初始时刻减少得较慢，但随着时间的推移，衰减速度加快。

规范化或平均化circles

  1. 计算总网格单元数：
    • MAX_RADIUS*2+1 计算了以当前像素为中心，波纹能影响到的网格单元的一维尺寸。由于波纹是在一个正方形区域内计算的，所以需要在两个方向上进行这个计算。
    • 将这个值乘以自己，即 (MAX_RADIUS*2+1)*(MAX_RADIUS*2+1)，计算出整个正方形区域内网格单元的总数。
  2. 规范化累加的波纹效果：
    • circles 变量在之前的代码中累积了来自所有相关网格单元的波纹影响。
    • 通过除以网格单元的总数，实际上是对这些影响进行了平均处理。这是一个规范化步骤，确保 circles 变量的值保持在合理的范围内，不会因为网格单元数量的增加而变得过大。
  • 这一步骤是重要的，因为它确保了不同区域的波纹影响是可比的，并且独立于 MAX_RADIUS 的具体值。这样可以使得效果在不同的参数设置下都能保持一致性。
  • 规范化处理有助于后续使用 circles 变量来计算片段的颜色和光照效果时，保持效果的稳定性和一致性。

计算光照强度

  • intensity 计算了一个基于时间变化的值，这个值在 0.01 和 0.15 之间变动。
  • smoothstep(0.1, 0.6, ...) 用于平滑地插值，它在两个值之间提供了一个平滑的过渡。
  • abs(fract(0.05*iTime + 0.5)*2.-1.) 创建一个随时间周期性变化的值。fract 函数取浮点数的小数部分，使得这个值在 0 到 1 之间循环。

波纹法线计算

  • 要使整个向量成为单位向量，z 分量需要满足这样的条件：x^2 + y^2 + z^2 = 1，其中 x 和 y 是 circles 向量的分量。
  • dot(circles, circles) 实际上计算的是 circles 向量的长度的平方，即 x^2 + y^2。
  • 通过从 1 减去这个值，我们得到 1 - (x^2 + y^2)，这正是 z^2 必须满足的条件，以确保整个向量的长度为 1。
  • 最后，使用 sqrt 函数取这个值的平方根，得到法线向量的 z 分量。
